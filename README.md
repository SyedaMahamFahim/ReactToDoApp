## Run Locally

Clone the project

```bash
  git clone https://github.com/SyedaMahamFahim/ReactToDoApp.git
```

Go to the project directory

```bash
  cd reacttodoapp

```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
